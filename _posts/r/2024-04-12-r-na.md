```{r}
library(tidyverse)
```

## ■ NA 여부 체크
### □ **anyNA**이용
```{r}
> anyNA(df)
[1] TRUE
```

<br>

### □ **is.na**이용
```{r}
> is.na(df)

```

<br>

### □ **table**이용
Data에 NA가 몇개 있는지 (TRUE 값)
```{r}
> table(is.na(df))

FALSE  TRUE 
   42     6
```

<br>

### □ **colSums **이용
각 컬럼별 NA 개수 보기
```{r}
> colSums(is.na(df))

 name class  math   eng 
    1     0     2     3 
```

## ■ NA 제거
### □ na.omit
NA가 포함된 모든 Data를 지울 수 있지만, 너무 많은 Data가 삭제될 수 있음
```{r}
> na.omit(df2)

      name class math eng gender
1 Hwayoung     1   74  76      f
7 Janghoon     1   55  70      m
```

<br>

### □ `tidyverse` 패키지 이용
#### ◎ **filter**
```{r}
# 수학 점수가 NA인 경우
> df %>%
+   filter(is.na(math))

      name class math eng
1      Sua     3   NA  77
2 Jaeyoung     3   NA  95
```

```{r}
# 수학 점수가 NA가 아닌 경우
> df %>%
+   filter(!is.na(math))

        name class math eng
1   Hwayoung     1   74  76
2       <NA>     2   56  70
3      Sojin     2   57  68
4   Minjeong     3   42  NA
5      Gaeun     1   39  63
6   Janghoon     1   55  70
7   Sunghoon     2   68  NA
8    Yeseung     3   83  NA
9    Jungjae     2   92 100
10 Kyunghwan     2   71  65
```

```{r}
# 수학과 영어 점수가 NA가 아닌 경우
> df %>%
+   filter(!is.na(math) & !is.na(eng))

       name class math eng
1  Hwayoung     1   74  76
2      <NA>     2   56  70
3     Sojin     2   57  68
4     Gaeun     1   39  63
```

#### ◎ **drop_na**
```{r}
> df %>%
+   drop_na()

       name class math eng

3     Gaeun     1   39  63
```

```{r}
df %>%
  drop_na(math)

df %>%
  drop_na(math,eng)
```

```{r}
#NA를 제거한 Data의 수학과 영어 평균점수
d> df2 %>%
+   drop_na(math, eng) %>%
+   group_by(class) %>%
+   summarize(mean(math), mean(eng))

# A tibble: 2 × 3
  class `mean(math)` `mean(eng)`
  <dbl>        <dbl>       <dbl>
1     1           56        69.7
2     2           69        75.8
```

<br>

#### ◎ **na.rm**
```{r}
> df2 %>%
+   summarize(mean(math, na.rm=T))

  mean(math, na.rm = T)
1                  63.7
```
