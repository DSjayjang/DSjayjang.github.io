# ※ Math
- 수학 계산과 관련된 라이브러리

<br>

## ■ 라이브러리 호출

```py
> import math
```

<br>

## ■ math

### □ .gcd
- 최대 공약수 (greatest common divisor) 출력

```py
# e.g.
> math.gcd(60, 100, 80)
# 20
```

### □ .lcm
- 최소 공배수 (least common multiple) 출력

```py
# e.g.
> math.lcm(60, 100, 150)
# 300
```
